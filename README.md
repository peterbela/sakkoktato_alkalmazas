# Sakkoktató alkalmazás kezdőknek

Ez a projekt egy webalapú sakkoktató alkalmazás, amely kezdő játékosok számára segít elsajátítani a sakk alapjait. Az alkalmazás leckéken, interaktív feladatokon, XP-rendszeren és szinteken keresztül vezeti végig a felhasználót.

## Fő funkciók

- Interaktív sakktábla megjelenítése
- Bábuk szabályos lépéseinek gyakorlása
- Leckék különböző sakkfigurákhoz és alapszabályokhoz
- Feladatok ellenőrzése a táblán végrehajtott lépések alapján
- XP-rendszer és szintlépés
- Leckék zárolása szint alapján
- Felhasználói adatok mentése SQLite adatbázisba
- Visszajelzés helyes és hibás megoldások esetén

## Használt technológiák

- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
